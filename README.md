# TezYet
Local taxi ordering system built with Django REST Framework


# TezYet Taxi API

TezYet — Sayram tumani uchun mo'ljallangan mahalliy taxi buyurtma platformasi.

Ushbu loyiha mijoz va haydovchilarni bog‘laydigan backend API bo‘lib, Django REST Framework yordamida yaratilgan.

## Features

* JWT Authentication
* Taxi order system
* Automatic price calculation
* Google Maps integration
* Driver location tracking
* Rating and review system
* Admin management system

## Tech Stack

Backend:

* Python
* Django
* Django REST Framework

Database:

* PostgreSQL

Other tools:

* Google Maps API
* JWT Authentication
* Swagger API documentation
